# Markdown 转 PDF

在 VScode 中编写笔记十分方便，但是有时我们需要将其转换为其他格式。最常用的格式是 `html` 和 `pdf`.

## 操作方法

由于我经常在 md 文档中插入 latex 语法，结合 pdf 转换需求，最终我选择了：

- 安装 Markdown all in one 拓展
- 将 .md 文件转换为 .html
- 在浏览器将 .html 文件 print 为 pdf（`Ctrl + P`）

P.S. 如果不想要页眉和页脚，在 print 的时候取消即可。
